# Product Overview

An e-commerce storefront application called "My Store" that displays products fetched from a Sanity CMS backend.

## Core Functionality

- Product catalog display with title, description, image, and price
- Sanity Studio embedded at `/studio` for content management
- Server-side data fetching using GROQ queries

## Content Model

- **Products**: Main content type with title, description, image, and price fields
